# ♟️ Desafio MateCheck - Nível Novato

Este repositório contém a solução do **Desafio de Xadrez - Nível Novato** proposto pela MateCheck.

## 📖 Descrição
O objetivo é movimentar três peças de xadrez utilizando **estruturas de repetição** em C:

- **Bispo**: 5 casas na diagonal superior direita (Cima + Direita).
- **Torre**: 5 casas para a direita.
- **Rainha**: 8 casas para a esquerda.

Cada peça foi programada utilizando **um tipo diferente de loop**:

- Bispo → `for`
- Torre → `while`
- Rainha → `do-while`

## 🛠️ Tecnologias
- Linguagem: **C**
- Compilador: GCC (ou compatível)
